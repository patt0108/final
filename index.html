Why should you hire Harvey Patterson?

I've been making websites in HTML and CSS since 1999. I'm an expert at HTML4 and CSS 1/2, and I'm becoming an expert in HTML5 and CSS3 at Algonquin College.

I started making interactive websites with JavaScript in 2000. I've made everything from calculators to pages that mimic Flash using JavaScript, and now the HTML5 canvas object has made JavaScript even more powerful.

Ready to move up to jQuery? Then I'm your guy. I learned jQuery while working for the Government of Canada in 2012, and when you find out how much time you can save having your coders use jQuery instead of JavaScript, you'll never look back.

I started building database-driven websites using mySQL in 2005. I learned IBM DB2 and PostgreSQL on the job in 2012, and SQLite at Algonquin College in 2015. There isn't a dialect of SQL that I can't adapt to!

I've been using PHP for server-side scripting since 2005. The most popular server-side language in the world, PHP 5 is more powerful than ever, and I'm learning PHP 7 as we speak (released on 3 December 2015!).

I learned the versatile Python language at Alqonguin College and I have used it both as an application language and as a server-side language. My final project in two courses was to turn my Raspberry Pi into a web server using an SQLite database and Python server-side code.
